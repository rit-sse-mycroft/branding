Mycroft

Mycroft is an extensible AI we built for the SSE lab

Inspired by Iron Man's J.A.R.V.I.S., we wanted to create an AI that could understand the people in the lab and respond to them interactively. We also wanted to let SSE members add new functionality in the future, and plug into other hardware and services as they become available.

Rapid Development Project

We built Mycroft during the 2014 winter break, spending 3 weeks working 8-9 hour days.

Rapid Development projects used to be built in the first weekend of winter and spring quarters, but the semester transition let us treat development like a real software project.

Tools

Mycroft was built to be extended on any platform, using any language with basic networking libraries. Here are some of the tools we used:

The core server, speech recognition app, and text-to-speech generator were all written in C#, and use the Microsoft Speech Platform.

Many of our development tools were written with Node.js, and we've created an NPM package for building Mycroft apps.

Ruby

Most of our plugins are built in Ruby, like our weather, jokes, and Pandora apps.

We used Python for our speaker and microphone apps, taking advantage of PyAudio for our multimedia handling.

Everything Mycroft can do is written in an "app" - a program running on any computer that talks to the central server using TCP sockets. Our communication API lets the core server direct apps to talk about each other, and apps respond when they can meet other applications' needs.

All of our code is open source and available online
